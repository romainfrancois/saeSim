Arguments

size
For sample_n, the number of rows to select. For sample_frac, the fraction of rows to select. If tbl is grouped, size applies to each group.
replace
Sample with or without replacement?
weight
Sampling weights. This expression is evaluated in the context of the data frame. It must return a vector of non-negative numbers the same length as the input. Weights are automatically standardised to sum to 1.
groupVars
character with names of variables to be used for grouping.

Sampling functions

Sampling functions

Description

These functions are intended to be used with sim_sample and not interactively. They are wrappers around sample_frac and sample_n.

Details

sample_numbers is a vectorized version of sample_number.

sample_cluster_number and sample_cluster_fraction will sample clusters (all units in a cluster).
