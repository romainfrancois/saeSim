saeSim

Tools for the simulation of data in the context of small area estimation. Go to the homepage of this project to find out more and read the vignette.

Installation

The package is actively developed. Please report bugs, troubles and discussion as issue on GitHub. You can install the latest version from the repository on GitHub:

library(devtools)
install_github("wahani/saeSim")

or from CRAN:

install.packages("saeSim")
